# aliengame
just a quick little game built to practice Vue and Vuex state management based on Sarah Drasner's alien zombie game

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
